# SQL-Based Analytics Dashboard (Power BI)

## Project Overview
This project is an end-to-end analytics dashboard built by connecting a MySQL database to Power BI using ODBC. The dashboard analyzes business performance through a well-structured data model and optimized DAX calculations.

## Tools & Technologies
- Microsoft Power BI
- DAX
- MySQL
- ODBC
- Data Modeling

## Data Modeling
- Designed a clean analytical model from a normalized SQL schema
- Implemented bridge tables and many-to-many relationships
- Optimized relationships for accurate aggregation and performance

## Dashboard Features
- KPI cards for key performance metrics
- Category and model-wise analysis
- Interactive slicers for dynamic filtering
- Detailed tables for drill-down analysis

## Key Business Insights
- Identified high-performing models and parts
- Highlighted distribution patterns across categories
- Enabled stakeholder-level analysis using interactive filters

## How to Use
1. Download the `.pbix` file
2. Open it in Power BI Desktop
3. Refresh data or explore insights using slicers
